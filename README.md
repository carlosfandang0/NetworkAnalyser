# NetworkAnalyser

A simple network scanner designed for Guition 480x480 display, developed with GitHub Copilot.

## About

This repository accompanies a technical presentation. It may serve as:
- An adjunct to an NMEA (National Marine Electronics Association) protocol talk
- The subject of a separate standalone presentation

## Overview

NetworkAnalyser is a network scanning tool optimized for embedded displays with 480x480 pixel resolution, specifically targeting Guition display modules.
